Time Qをインストールしていただきありがとうございます！

【説明】
放送や配信で使える、オフライン専用のタイムキーパーアプリです。
ディレクターとパーソナリティが同じWi-Fi（無線LAN）環境に接続することで、リアルタイムに進行や指示を共有できます。

【Wi-Fi環境の作り方】
以下のいずれかの方法で、ディレクターとパーソナリティが同じWi-Fiに接続してください。

方法1： 既設のWi-Fiルーターに接続する。

方法2： ディレクターのWindows PCをWi-Fi親機にする。（モバイル ホットスポット機能）

方法3： ディレクターのMacをWi-Fi親機にする。（インターネット共有機能）

【使い方】

ディレクター役の人が、PC/Macでこのアプリを起動し、「部屋を作る」ボタンを押します。

画面に表示されるQRコードと、IPアドレスを含んだURL（http://...）を確認します。

パーソナリティ役の人は、ディレクターと"同じWi-Fi"にスマートフォンやiPad、別のPCを接続します。

パーソナリティはスマートフォンのカメラでQRコードを読み取るか、ブラウザでURLを直接入力してアクセスします。

これで接続は完了です！


【動作環境】
サーバー（ディレクター役）： Windows / macOS
クライアント（パーソナリティ役）： PC, Mac, iOS, Androidの各種Webブラウザ
